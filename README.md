# Serkan Bayram | Engelsiz ve Erişilebilir Dünya Platformu

Bu proje, Milletvekili Serkan Bayram liderliğinde engelli hakları savunuculuğu ve küresel erişilebilirlik vizyonunu temsil eden modern bir web platformudur.

## Temel Özellikler
- **Erişilebilirlik Motoru:** Dinamik font ölçeklendirme, yüksek kontrast filtreleri ve disleksi dostu mod.
- **Yapay Zeka Entegrasyonu:** İletişim mesajlarını otomatik kategorize eden Genkit akışları.
- **Modern UI:** Tailwind CSS ve ShadCN bileşenleri ile tamamen duyarlı tasarım.
